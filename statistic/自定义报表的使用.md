对面mate表 以下为各字段使用说明
field 为 api访问所学要用到的标识符
例如:api/adm/stat/meta/test 
     此处标识符为 test         field 中填 test
query_sql 中填需要查询的sql
type 中填查询出来的字段 对应的类型简称 必须字段-字段类型 一一对应
search 中填需要查询的字段 不在查询出来的字段中有的需要标注类型 例如 时间-T

 


类型定义
    public static final String DECIMAL="D";//金钱
    public static final String TIME="T";//时间
    public static final String PERCENT="P";//百分比
    public static final String COUNT="C";//数量
    public static final String STRING="S";//字符串
@if标签使用：
在sql任意位置 以@if开始@end结束
@if[判断条件] 可用符号：&& || ( ) + - * / > < == !=  暂不支持<= >=
    此处填充sql 需要替换的变量用#{变量}
@end
字符类型:S
区段类型：T,D,C,P
区段类型的使用：
 @if[时间$left!=null&& 时间$right!=null] 							
	and o.created_date > #{时间$left}
    and o.created_date < #{时间$right}
@end
字符类型的使用
@if[盟友!=null]
   and a.alliance_name like %#{盟友}%
@end
